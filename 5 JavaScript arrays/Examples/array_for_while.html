Practical Recommendations
forEach() - DEFAULT CHOICE

Use when you need to perform operations on each element without early exit.

countries.forEach(country => console.log(country));
for...of - Early Termination

Use when you need break or continue based on conditions.

for (const country of countries) { if (condition) break; }
Traditional for - Index Operations

Use for complex index calculations or maximum performance with large arrays.

for (let i = 0; i < arr.length; i++) { arr[i] *= 2; }
for...in - NEVER for Arrays

Includes inherited properties, string keys, and order is not guaranteed.

// AVOID: for (let key in array) { ... }
