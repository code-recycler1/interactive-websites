JavaScript is a scripting language that is widely used to make web pages interactive and to develop web applications. The syntax of JavaScript shows similarities with the programming language Java.

Events form the heart of a JavaScript page. Event handling allows JavaScript instructions to be executed at the right moment, when the right event has taken place: A calculation is performed when a user clicks a button, a figure changes when the user moves the mouse over it, an animation starts when the page has loaded...

In JavaScript, a function can be used to execute a group of instructions under a given name. The instructions or statements inside the function are executed when the function is called. A function can receive values and return them to the point of the call.
