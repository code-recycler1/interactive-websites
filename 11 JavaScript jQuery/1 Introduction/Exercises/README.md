# JavaScript jQuery introduction - Exercises

## Exercise 1

Create a sample page to test whether jQuery works. Do this, for example, by executing an alert() within $(document.ready()).

## Exercise 2

Create a page with a number of divs. Give these divs a blue background colour and white foreground colour (text colour) when the page loads.

## Exercise 3

Create a table on the page. Give the odd rows of the table a light grey background colour.
