# JavaScript jQuery events - Exercises

## Exercise 1

In the script events_09.html, the mouse position is shown for the entire page. Adapt this script so that the mouse position is only shown when the mouse hovers over a specific div. This div gets a height and width of 400px via CSS.

## Exercise 2

Ensure that all text in a text field is selected when the visitor clicks in the text field (so that they can easily copy the text. Also see YouTube for embedding videos).

## Exercise 3

Write a script that captures the Enter key in a text field. For normal letters, the code is displayed in a div; for Enter, an alert box should appear with something like “You pressed Enter, we are going to submit the form!”.

## Exercise 4

Create a page with a text field where the visitor can enter a number. This number of items is added to a list. Ensure that on mouseover over each list item, a tooltip appears indicating which item the mouse is currently hovering over.
